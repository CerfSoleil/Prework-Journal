Git

  1. "ssh -T git@github.com" will establish and test your connection to GitHub
  2. "git status" will check what branch you are currently on as well as list untracked changes or additions to the files.
  3. "git checkout -b BRANCH/NAME" will create a new branch and switch to it.
  4. "git add -A" will add the modification in the current working branch to the staging area. "-A" indicates that we want to add all changes.
  5. "git pull origin main" will recieve main or a branch's modifications into the local working area.
  6. "git push origin BRANCH/NAME" will push the branch to GitHub in order for you to share or create a pull request.
  7. "git commit -m "COMMIT MESSAGE" will update the repository with your local changes.
  8. The Pull Request must then be handled on GitHub.
